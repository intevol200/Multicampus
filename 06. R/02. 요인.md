## 요인의 개념
* 요인은 벡터의 한 종류
* 요인은 주로 집단별로 자료를 분석하고자 할 때 특정 자료를 범주형 자료로 변경해 줌
    - 벡터 : 범주형 자료로 인식되지 못함
    - 요인 : 범주형 자료로 인식
        + 명목형 : 성별, 국적
        + 순서형 : 매우 만족, 만족, 불만족
```r
gender_f2 = factor(gender, levels = c("m", "f"),
                   labels = c("남자", "여자"))
#결과
#남자 여자 여자 여자 남자
#Levels: 남자 여자


#빈도수 계산
gender_f2 = factor(gender, levels = c("m", "f"),
                   labels = c("남자", "여자"),
                   ordered = TRUE)

sort(gender_f2)
#결과
#남자 남자 여자 여자 여자
#Levels: 남자 < 여자
```
